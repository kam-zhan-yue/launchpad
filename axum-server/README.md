# An Axum Webserver

A basic template to jumpstart an axum webserver with as little dependencies as possible.
| Tool | Description |
| - | - |
| Bacon | Running and testing the dev environment |
| Mise | Task and tooling management |
| Docker | CI/CD |

```bash
mise trust
mise install
mise start # for local instance
mise docker # for docker instance
```
