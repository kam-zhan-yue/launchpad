# React

An opinionated React framework. Feel free to add and remove packages depending on your use-cases.

| Packages | Usage |
| - | - |
| Tanstack Router | Routing and rendering purposes |
| Tanstack Query | Querying state management and caching data |
| Three JS | 3D animation and rendering |

```shell
mise trust
mise install
mise start
```
